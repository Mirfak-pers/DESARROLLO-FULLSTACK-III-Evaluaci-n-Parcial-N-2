
# Microservicio de Inventario - SmartLogix

## 1. Descripción general

Este microservicio forma parte del proyecto **SmartLogix**, una plataforma orientada a la gestión logística para eCommerce.
El microservicio de Inventario permite administrar productos, consultar el stock disponible, actualizar cantidades, descontar stock cuando se genera un pedido y registrar movimientos básicos de inventario.
Este componente fue desarrollado como parte de la Evaluación Parcial N°2 de la asignatura **Desarrollo Fullstack III**.

---

## 2. Objetivo del microservicio

El objetivo principal del microservicio es permitir la gestión del inventario de productos de SmartLogix, entregando endpoints REST que puedan ser consumidos posteriormente por el Backend For Frontend y por el frontend del sistema.s
Con este servicio, los usuarios podrán visualizar productos y stock disponible desde la interfaz web.
